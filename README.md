# Anti_f5_attack
You can implement Anti-f5-attack using PHP.

<h2>How to use</h2>
Please only write this in first of php file.  

### ↓before↓ ###
    <!DOCTYPE html>
    <html>
      <body>
        <p>Text</p>
      </body>
    </html>
    
### ↓after↓ ###
    <?php
      require "./anti_f5.php";
    ?>
    <!DOCTYPE html>
    <html>
      <body>
        <p>Text</p>
      </body>
    </html>
